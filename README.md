#emacs shortcut basic(emacssortcut.txt has more)

커서이동 
#
Ctrl f, b, n, p (화살표와 동일)  
Ctrl v, Alt v (PgUp, PgDn)  
Ctrl a, Ctrl e (Home, End)  
Alt f, b (단어단위로 앞뒤이동)  
Alt a, e (문단 앞, 뒤로 이동)  
Alt <, > (파일의 처음, 끝으로 이동)  
Alt r (커서를 페이지의 처음,끝,가운데로 이동)  
Ctrl l (커서가 포함된 줄을 화면 처음, 끝, 가운데로 이동)  
Alt g - Alt g (원하는 줄로 이동)
